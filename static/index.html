How to use the .ics file in Outlook

Teacher

  1. Go to outlook.com and sign in.
  2. Open Calendar → Add calendar → Upload from file and select the downloaded .ics file.

Students (attendees)

Share the same .ics file with your students (email, Teams, etc.). When they open the file (double‑click or open attachment), Outlook adds the classes to their calendar. No separate file per student.
